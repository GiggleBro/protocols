1. 适用范围

本协议适用于所有下载、安装、注册和使用本应用的用户。本应用提供的AI生成服务可能需要订阅或付费解锁，具体服务内容以应用内说明为准。

2. 订阅服务

2.1 订阅内容

订阅用户将获得以下权益：

  • 使用所有AI文案生成功能（改写/标题/种草/续写等）
  • 高清视频文案提取（支持4K视频）
  • 图片文字识别功能
  • 优先使用新推出的AI模板
  • 去除所有广告展示
  • 专属客服支持

2.2 订阅周期与费用

提供月度/季度/年度三种订阅周期，具体价格以应用内显示为准。付款将通过您的Apple ID或Google Play账户扣款。

订阅将在每个周期结束前24小时自动续订，除非您主动取消自动续订。

2.3 订阅管理

您可随时通过以下方式管理订阅：

  • iOS用户： [设置 → Apple ID → 订阅]
  • Android用户： [Google Play商店 → 订阅]

取消订阅后，您仍可享受订阅权益至当前周期结束。
