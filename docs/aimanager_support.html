水印工厂技术支持

欢迎来到 水印工厂 技术支持中心！如果您在使用过程中遇到任何问题，请参考以下常见问题，或联系我们的技术支持团队。

📌 常见问题（FAQ）

🔹 如何管理或取消订阅？

您可以通过：
iOS设备： [设置 → Apple ID → 订阅]
Android设备： [Google Play商店 → 订阅] 来管理或取消您的订阅。

🔹 如何提取视频中的文案？

两种方式：
1. 视频链接提取： 在【视频文案】页面粘贴视频链接，点击【开始解析】
2. 本地视频提取： 在【本地视频】选择相册视频文件，点击【开始解析】

🔹 图片识别文案不准确怎么办？

请尝试：

  • 确保图片清晰度高、文字方向正确
  • 调整图片亮度/对比度后再识别
  • 手动修正识别结果（支持编辑功能）

🔹 AI文案生成如何使用？

操作步骤：

  • 在【文案创作】选择需要的功能（改写/标题/种草等）
  • 输入原文或关键词
  • 点击【生成】获取多个版本结果
  • 可收藏或导出满意结果

🔹 遇到功能异常怎么办？

建议尝试：

  • 检查网络连接是否正常
  • 清除应用缓存（设置 → 应用管理）
  • 更新到最新版本
  • 重启设备
